Le pangramme

Introduction

Un pangramme est "une phrase comportant toutes les lettres de l'alphabet." (wikipedia)

Objectif

  • Maîtriser la création de documents dans Microsoft Word

  • Se familiariser avec tous les caractères, y compris les caractères accentués

  • Déposer un fichier sur un espace dédié en ligne (Dossier partagé sur Google Drive)

Ressources

http://info.sio2.be/infobase/5/9.php

Consignes

  • Créez un nouveau document "Word"

  • L'enregistrer (Bien choisir l'emplacement et le nom du fichier)

  • Choisissez 2 pangrammes parmi les textes proposés :

  • Recopiez-le.

  • Pour les caractères non disponibles directement sur le clavier, allez chercher l'outil adapté dans : "Insertion=>Symboles"

Une fois l'exercice fini...

  • Déposer le fichier dans le dossier partagé Google Drive
